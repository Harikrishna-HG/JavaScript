JavaScript Regular Expressions

A regular expression is a sequence of characters that forms a search pattern.

The search pattern can be used for text search and text replace operations.

    
Syntax

    /patterns/modifiers;

    Example

    /researchs/i;

    Explained explained :

    /researhs/i is a regular expression
    researchs is a pattern ( to be used ina search).
    i is a modifier (modifies the search to be case-insensitive).

Using String Methods

In JavaScript, regular expressions are often used with the two string methods:

search()

replace()

Using String search() with a string

The search() method searches a string for a specified value and returns the position of the match

Example

  let text = "Welcome to the programming!";
  let n = text.search("the");

  
The result in n will be

  
11

Using String search() with a Regular Expression

  let text = "Welcome to the programming!";
  let n = text.search(/The/i);

  
The result in n will be

  
11

Using String replace() With a String

The replace() method replaces a specified value with another value in a string:

let text = "Welcome Microsoft!";
let result = text.replace("Microsoft", "Google");
 
The result will be:

 
Welcome Google

Use String replace() With a Regular Expression

  let text = "Visit Microsoft!";
let result = text.replace(/microsoft/i, "Google");

The result will be:


Welcome Google

JavaScript Modules

JavaScript modules allow you to break up your code into separate files.

This makes it easier to maintain a code-base.

Modules are imported from external files with the import statement.

Modules also rely on type="module" in the <script> tag.

Export

Modules with functions or variables can be stored in any external file.

There are two types of exports:

  1. Named Exports
  2. Default Exports
