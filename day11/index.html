The Object Datatype

The object data type can contain both built-in objects, and user defined objects:

Built-in object types can be:

objects, arrays, dates, maps, sets, intarrays, floatarrays, promises, and more.

    // Numbers :

    let length = 16;
    let breadth = 18;

    // Strings

    let color = "Green";
    let name = "Harikrishna"

    // Booleans

    let x = true;
    let y = false;

    // object

    const person = {
      firstName : "Harikrishna",
      lastName : "Gautam"
    };

    // Array object

    const cars = ["BMW", "Volvo", "Tesla", "Bugatti"];


    // Data object :

    const date = new Date("2022-03-25");

JavaScript Booleans

Booleans can only have two values: true or false.

    let x = 5;
    let y = 5;
    let z = 6;
    (x == y) // Returns true
    (x == z) // Returns false

JavaScript Types are Dynamic

JavaScript has dynamic types. This means that the same variable can be used to hold different data types: 

    let x; // Now x is undefined
    x = 5; // Now x is a Number
    x = "John"; // Now x is a String

JavaScript Arrays

JavaScript arrays are written with square brackets. 

    const cars = ["Saab", "Volvo", "BMW"];

The typeof Operator

We can use the JavaScript typeof operator to find the type of a JavaScript variable.

    typeof ""             // Returns "string"
typeof "Harish"         // Returns "string"
typeof "Harish Gautam"     // Returns "string"

Undefined

In JavaScript, a variable without a value, has the value undefined. The type is also undefined. 

    let car;    // Value is undefined, type is undefined
