JavaScript Arithmetic

In algebra, we can do arithmetic with JavaScript variables, using operators like = and +:

  let x = 5 + 2 + 3;

We can also add strings, but strings will be concatenated:

let x = "5" + 2 + 3;

JavaScript Dollar Sign $

let $ = "Hello World";
  let $$$ = 2;
  let $myMoney = 5;

JavaScript Underscore (_)

  let _lastName = "Johnson";
  let _x = 2;
  let _100 = 5;
