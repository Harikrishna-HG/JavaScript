Day 8 of learning JavaScript

JavaScript Const

The const keyword was introduced in ES6.

1.Variable cannot be reassigned.

2.Variable defined with const cannot be redeclared.

3.Variable defined with const have blockscope

Use of const when we declare.

1. A new Array

2. A new Object

3. A new Function

4. A new RegExp

We can change the elements of constant array.

Change the properties of the constant object.

Constant Arrays

    /* We can create an constant array */
    const cars = ["BMW","Mercedes","Tesla"];

    /* We can change an element */
    cars[0] = "Lamborghini";

    /* We can add an element */
     cars.push("Audi");

Constant Objects

  /* We can change the properties of a constant object */

  const car = { type:"fiat", model:"500", color:"White"};

  /*  We can change the property */

  car.color = "red";

  /* We can add a [property] */

  car.owner = "Harikrishna";
