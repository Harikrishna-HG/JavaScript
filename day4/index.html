Day 4 of Learning JavaScript

JavaScript Statements are composed of the:

Values, Operators, Expressions, Keywords, and Comments

JavaScript statements are executed one by one, in the same order as they are written.

For best readability, programmers have to avoid code lines longer than 80 characters.

Best practice is to break the code after an operator.

Multiple statements on one line are allowed:

  a = 20; b = 6; c = a + b; //separated by semicolon
    
  a = 20; // Declare of variables
  b = 20;
  c = a + b;
